# remnanode-online-users

`remnanode-online-users` — простой bash-скрипт для подсчёта активных пользователей RemnaNode/Remnawave-ноды по `access.log`.

Скрипт читает `/var/log/remnanode/access.log`, достаёт из строк лога `user_id` и IP-адрес, после чего считает активных пользователей за выбранный промежуток времени.

По умолчанию считает статистику за последние **60 секунд**.

---
